Calculadora de Média em R

Este é um exemplo de um programa simples em R que calcula a média de um conjunto de números fornecidos pelo usuário. O programa solicitará ao usuário que insira os números separados por vírgula e, em seguida, calculará a média.

# Função para calcular a média
calcular_media <- function(valores) {
  valores <- as.numeric(strsplit(valores, ",")[[1]])
  media <- mean(valores)
  return(media)
}

# Solicitar entrada do usuário
valores <- readline(prompt="Digite os números separados por vírgula: ")

# Calcular a média e imprimir o resultado
media <- calcular_media(valores)
cat("A média dos números fornecidos é:", media, "\n")

Explicação do Programa

Neste programa, definimos uma função chamada calcular_media que recebe uma string de números separados por vírgula, converte-os em um vetor numérico e calcula a média. Em seguida, solicitamos ao usuário que insira os números, chamamos a função calcular_media e imprimimos o resultado.
