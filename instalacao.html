Passo a Passo para Instalar o R

  1. Baixar o R

    Visite o site oficial do R em CRAN.

  2. Escolher seu Sistema Operacional

    Selecione o link de download apropriado para o seu sistema operacional (Windows, macOS ou Linux).

  3. Instalar no Windows

    • Baixe o instalador do R para Windows.
    • Execute o instalador e siga as instruções na tela.
    • Escolha as configurações padrão para a maioria dos usuários.

  4. Instalar no macOS

    • Baixe o arquivo .pkg do R para macOS.
    • Abra o arquivo e siga as instruções na tela para instalar.

  5. Instalar no Linux

    • Abra o terminal.
    • Adicione o repositório do CRAN ao seu sistema de pacotes.
    • Atualize a lista de pacotes do sistema.
    • Instale o R usando o gerenciador de pacotes do seu sistema.
    # Comandos para Debian/Ubuntu
sudo apt update
sudo apt install r-base
# Comandos para Fedora
sudo dnf install R
# Comandos para Arch Linux
sudo pacman -S r

  6. Verificar a Instalação

    Abra o VSCode, vá em extensões digite R e baixe a primeira.

  7. Teste

    • Crei um arvio de test .R e escreva um programa básico como um print("Hello World"), se a mensagem aparecer no Prompt quer dizer que funcionou.
